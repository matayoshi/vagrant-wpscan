wpscanを試す
============

wpscanを試すためのVagrantfile一式

## Description

wpscanを試すためのVagrantfile一式です。  
Ubuntu 14.04(Trusty Tahr) の環境にwpscanコマンドをインストールします。

## Requirement

- Vagrant
- VirtualBox
- Ansible
- Memory > 4GB

## Usage

```bash
$ vagrant up
$ cd wpscan
$ ./wpscan.rb
```
